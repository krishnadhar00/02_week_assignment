Ordered list

  1. html tag:

    Defines the root of an HTML document

  2. head tag:

    It contains the meta/information for the document

  3. Anchor tag:

    Defines a hyperlink

  4. Bold tag:

    It bold the text.

  5. Italic tag:

    It italic the text.

  6. Underline tag:

    It underlined the text

  7. Span tag:

    It is used for styling the specific text

  8. table tag:

    It defines the table

  9. Div tag:

    It is used for grouping

  10. image tag:

    Defines an image
