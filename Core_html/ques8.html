Frontend Tech Stack: For most of the web applications, the choice of frontend remains similar. Modern web applications are based on HTML, CSS, JavaScript, TypeScript, or frameworks based on JS, like React and Express.These are used for frontend.
Html : HTML (Hypertext Markup Language) defines the basic layout and structure of your pages. For example, the sections, bulleted points, and tables are created using HTML tags.
CSS : All the styling that's applied to the web pages comes from CSS—for example changing the background-color of web page.
JAVASCRIPT : The dynamic elements of the page come from the JavaScript code written by developers. For example, what happens when you hover your mouse over an image, or when you click on a submit, like, or comment button is decided by JavaScript events in the code.
There are many other frameworks and libraries—like Bootstrap, Vue, React, Angular, and jQuery—that are built on JavaScript and provide additional functionalities and common use cases
Backend Tech Stack : Crunching and munching of data can be done in backen part. Backend contains the logic or algorithm part to code.It contains express.js,node.js,angular.js etc. It is used for backend part.
Express.js : Express is a node js web application framework that provides broad features for building web and mobile applications. It is used to build a single page, multipage, and hybrid web application.
Node.js : Node. js is fast and lightweight, and its ability to employ JavaScript code on both the frontend and backend opens up new development possibilities.
Data Storage Level tech Stack : It Stores the data for the future use.
MongoDB : The MongoDB database has a flexible data model that enables you to store unstructured data, and it provides full indexing support.
Full stack tech stack : This is the most common type of stack used in desktop web applications. It consists of both frontend and backend technologies—for example, Angular for the UI, a framework like Django for the application logic, and a database like MongoDB for managing data.
